# Conway's Game of Life GACNNs

**GA-optimized CNNs for predicting Conway's Game of Life**

![Rust 1.84.0 badge](https://img.shields.io/badge/Rust-1.84.0-E33B26)

This repository contains a solution for training and optimizing CNNs using GA to predict 8Ã—8 toroidal boards of Conway's Game of Life any number of timesteps in the future. The final trained CNN will be saved for standalone use.
